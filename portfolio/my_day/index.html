My Day

In this school project, developed with 6 friends, we had to build a web interface displaying several information from different APIs.

We divided the project in 3 parts :

  1. Frontend : creation of the web interface
  2. Backend : creation of the server
  3. Data : call to APIs and processing of the collecting data

We used a MERN (MongoDB, Express JS, React JS and Node JS) Stack. As I was already familiar with React, I decided to be versatile and I contributed to both the backend and the data parts and I helped in the frontend when needed.

We decided to build a single page interface displaying useful information that a user would want in the morning before starting his day. We created different widgets for the different information.

The user has to connect with his Google account. In order to achieve this, we used the Google Sign In API. We have a weather widget with information from the OpenWeatherMap API an air quality widget with information from the AQICNAirNetSensor Network API. We developed a simple algorithmn that advice an outfit based on the weather and we displayed the adviced outfit on the weather widget. We then have a calendar widget that show the upcoming events of the user thanks to the Google Calendar API. Finally, we have a widget that display the upcoming event of the user on a map and displayed the best mean of transport as well as the adviced hour of departure based on the weather, the air quality, the distance and the time to go there depending on traffic thanks the Google Directions API.
